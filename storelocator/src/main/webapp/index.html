Spring MVC Feature Demonstration

The URL, ./demo/yourID/feature?msg=Msg_From_Me, will be parsed by Spring MVC controller once clicked.

The URL, ./demo/yourID/feature?msg=exception, will trigger the exception handler.

The URL, ./TestDB.jsp , verifies the JDBC and JNDI resource configuration.

The URL, ./store/mystore , demonstrates how to use google map to locate stores nearby.
